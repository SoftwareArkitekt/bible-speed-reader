This is my own project to develop a speed reading application for Bible translations that combines all of the features that I felt were missing in other tools. Features, in no particular order:

* ORP (Optimal Recognition Point) text alignment
* Customizable colors, including preset Dark/Light/Sepia themes
* Custom speed settings
* Word complexity delay settings
* Proper noun complexity delay settings
* Progress meter + local storage of progress

For input files, see https://github.com/Amosamevor/Bible-json/tree/main/versions/en
